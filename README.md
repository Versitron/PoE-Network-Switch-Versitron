## Frequently Asked Questions

### What is a PoE+ network switch?  
A PoE+ network switch is a networking device that delivers both Gigabit Ethernet connectivity and enhanced Power over Ethernet (up to 30W per port) to connected devices.

### What devices can be powered by PoE+?  
PoE+ can power a wide range of devices including IP cameras, wireless access points, VoIP phones, and other PoE-enabled endpoints requiring higher power.

### Do PoE+ switches support standard PoE devices?  
Yes PoE+ switches are backward compatible with IEEE 802.3af (standard PoE) devices.
